## 💡 ドキュメント作成のルール

新しいドキュメントを追加する際は、以下のルールに従ってください:

1. **マークダウン形式** (`.md`) を使用
2. **絵文字とセクション分け** で読みやすく
3. **INDEX.md を更新** して新ドキュメントへのリンクを追加
4. **日本語で記述** (コード例は英語OK)
5. **更新日を記載** して情報の鮮度を明示
